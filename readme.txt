Clone the Sample Project

Clone this repo to your computer using git, and cd into the project directory.

git clone https://github.com/stormpath/express-stormpath-sample-project.git
cd express-stormpath-sample-project
Install Dependencies

Install the dependencies from package.json.

npm install
Set Environment Variables

Configure the sample project by using environment variables.

SELECT APPLICATION
export STORMPATH_CLIENT_APIKEY_ID=4RK74S7EA882UKPF3CL7JLVST
export STORMPATH_CLIENT_APIKEY_SECRET=N4j0IU3sfOTsaErVJ0FVJZZqYIA5SuLPorDAYOOThLA
export STORMPATH_APPLICATION_HREF=https://api.stormpath.com/v1/applications/6Auftj9giDcNjBWMvFCSSX
Note: This API key has been autogenerated for this QuickStart. Please generate a new one for use in production.
Start the Server

Start the server by running the following command:

node server.js
That's It

You now have a boilerplate application with Stormpath already integrated. Browse to http://localhost:3000/ and try it out!
